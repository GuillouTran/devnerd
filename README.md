## **Installation 💪🏼**

1.  Clone project

        git clone https://github.com/ahendouz/react-redux-todo.git

2.  cd into folder and download dependencies and starting dev server

        cd react-redux-todo
        yarn

3.  Starting dev server

        yarn start
